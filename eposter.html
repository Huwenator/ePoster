Some tips for you...

  • Posters need to be created in a 16:9 aspect ratio
  • Recommended poster dimensions are 3840px x 2160px (1920px x 1080 minimum)
  • Sample PowerPoint file can be downloaded from here
  • Smaller sizes will still display but the zoom function may not work as expected
  • Posters should be submitted as either a .png or .jpg file
  • Poster can be submitted via a custom online form such as this one -
    Submit your poster
  • Top left logo is customisable as are the screen colours but netural tones work best
  • Works best in Google Chrome with a monitor resolution of 1920 x 1080
  • Search by poster number, name, author or keyword
  • Esc to exit fullscreen mode
